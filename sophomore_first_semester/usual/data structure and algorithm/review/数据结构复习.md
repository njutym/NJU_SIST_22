## 4.0 树

大结构必考

概念过一下，考概念应用

+ binary tree数组表示，链接实现都要掌握
+ 树的遍历必要，先序中序后序建树（正反）
+ maketree基本的也会考
+ 递归算法要会写，非递归读得懂
+ 字符串赋值、取子串、判断相等
+ 树的存储。广义表一般不考，双亲（并查集）和左右兄弟要考
+ 树和森林的遍历
+ 线索化树经常考
+ 霍夫曼树有时会考，建议看一下
